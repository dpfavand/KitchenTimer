# Two-phase looping timer

For recipes where you repeate the same two timer lengths to work through the entire batch. For example:

## Belgian Gallets
Such as documented here: https://feasteveryday.blogspot.com/2018/12/gallets-belgian-waffle-cookies.html
I like to have a timer automatically looping for 1 minute for cooking, and 15 seconds for opening the iron, removing the cookies, setting the new dough, and closing the iron. While the fingers get a bit messy, this stays on a pretty tight schedule. An automatic looping timer means I don't have to fiddle with the phone or set a timer for every pair of cookies.

## Pancakes
I generally cook buttermilk pancakes for 3 minutes on the first side and 2 minutes on the second. Maybe this needs a third period of 30 seconds to get them off the griddle and pour more?
